# مثال: امتیازدهی ممپول و مونتاژ بلاک

فرض کنید ممپول دارای تراکنش‌های (txs) A, B, C, D با یال‌های A→C, B→C, C→D است.

## امتیازات (توضیحی)
- کارمزد/اندازه:
  - A: 200/400 = 0.5
  - B: 250/250 = 1.0
  - C: 150/200 = 0.75 (اجداد A,B)
  - D: 100/150 = 0.67 (جد C)

### نرخ کارمزد اجدادی
- A: (200)/400 = 0.50
- B: (250)/250 = 1.00
- C: (200+250+150)/(400+250+200) = 600/850 ≈ 0.71
- D: (200+250+150+100)/(400+250+200+150) = 700/1000 = 0.70

مرتب شده بر اساس نرخ اجدادی: B, C, D, A.

## انتخاب حریصانه با ظرفیت ثابت
- ظرفیت بلاک ۱۰ تراکنش شامل کوین‌بیس است.
- B را انتخاب کنید.
- سپس C را در نظر بگیرید (نیاز به A و B دارد): اگر ظرفیت اجازه دهد A+B+C را شامل شود.
- سپس D را در نظر بگیرید (نیاز به A+B+C دارد): اگر ظرفیت اجازه دهد، شامل شود.

## ترتیب توپولوژیکی زیرگراف انتخاب شده
- یک ترتیب معتبر: A, B, C, D (والدین قبل از فرزندان)

## مونتاژ بلاک
- کوین‌بیس ابتدا (به سمت ظرفیت محاسبه می‌شود)
- سپس A, B, C, D به ترتیب توپولوژیکی
- ریشه مرکل را محاسبه کنید، نانس را استخراج کنید، خروجی بلاک JSON.
