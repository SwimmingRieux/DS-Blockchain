# دستورات CLI

## GiveKeyPair
- خروجی: `privateKey publicKey` (هگزادسیمال)، با فاصله از هم جدا شده

## SetDifficulty x
- تنظیم تعداد بیت‌های صفر پیشرو مورد نیاز به `x`

## AddTransactionToMempool txJson
- تجزیه و اعتبارسنجی تراکنش؛ پذیرش به ممپول
- خروجی: ممپول مرتب شده به ترتیب نزولی بر اساس کارمزد اجدادی به ازای هر بایت

## EvictMempool x
- حذف `x` تراکنش کم‌ارزش‌تر بر اساس کارمزد فرزندان به ازای هر بایت (با حذف فرزندان آنها نیز)
- خروجی: ممپول مرتب شده به ترتیب صعودی بر اساس کارمزد فرزندان به ازای هر بایت

## MineBlock
- انتخاب بر اساس کارمزد اجدادی به ازای هر بایت تا سقف اندازه بلاک، مرتب‌سازی توپولوژیکی، محاسبه ریشه مرکل، انجام اثبات کار (PoW)، ایجاد کوین‌بیس، به‌روزرسانی UTXO و ممپول
- خروجی: JSON کامل بلاک

## GetMerkleProof txid height
- خروجی: `{ siblings: [hex...], positions: ["L"|"R"...] }`

## VerifyTxInclusion txid height proofJson
- خروجی: `true` یا `false`
